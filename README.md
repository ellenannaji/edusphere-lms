
# 📚 EduSphere - Full Stack Learning Management System (LMS)

EduSphere is a full-stack web-based Learning Management System built using Flask and SQLite.  
It provides a complete e-learning platform with courses, video lessons, quizzes, progress tracking, and certificate generation.

---

## 🎯 Project Overview

EduSphere allows students to:
- Register and login securely
- Enroll in courses
- Watch embedded video lectures
- Attempt quizzes
- View performance results
- Download certificates

This project demonstrates full-stack web development using Python Flask.

---

## ✨ Features

### 👤 Authentication System
- User registration & login
- Session-based authentication

### 📚 Course Management
- Multiple courses available
- Course details with description and instructor

### 🎥 Video Learning
- Embedded YouTube videos per course
- Structured learning flow

### 📝 Quiz System
- MCQ-based quizzes
- Automatic score calculation
- Instant result display

### 📊 Progress Tracking
- Score tracking per user
- Percentage calculation

### 🎓 Certificate System
- PDF certificate generation
- Available for students scoring 70%+
