# 📚 EduSphere - Online Learning Management System (LMS)

EduSphere is a full-stack web-based Learning Management System built using **Flask + SQLite**.  
It provides students with a complete e-learning experience including courses, video lectures, quizzes, progress tracking, and certificates.

---

## 🚀 Features

- 🔐 User Registration & Login system
- 📚 Course Management
- 🎥 Video Learning system (YouTube embedded)
- 📝 MCQ Quiz system with auto evaluation
- 📊 Quiz results with score & percentage tracking
- 🏆 Progress tracking system
- 🎓 Certificate generation (PDF download)
- 📈 Student dashboard

## 🎯 Key Modules

### 👤 Authentication
Users can register, login, and maintain sessions.

### 📚 Courses
View and access multiple learning courses.

### 🎥 Video Learning
Embedded YouTube videos for each course.

### 📝 Quiz System
MCQ-based quiz with automatic scoring.

### 📊 Dashboard
Shows:
- Courses
- Quiz results
- Performance percentage

### 🎓 Certificate System
Students can download certificate after scoring 70%+.
